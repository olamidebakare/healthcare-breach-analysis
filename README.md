# US Healthcare Data Breach Analysis

## Overview
An analysis of healthcare data breaches reported to the U.S. Department of 
Health and Human Services (HHS) Office for Civil Rights (OCR), exploring 
patterns, trends, costs, and security implications.

## Key Findings
- Healthcare breaches cost an average of $7.42 million per incident (2025)
- 279 days average time to identify and contain a healthcare breach
- Hacking accounts for 90%+ of breach incidents
- Over 276 million patient records compromised in 2024 alone

## Tools Used
- Python (pandas, matplotlib, seaborn)
- Jupyter Notebooks
- Power BI
- SQL
